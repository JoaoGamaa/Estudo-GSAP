01 / Base

Fundamentos do GSAP

gsap.to()

Animar do estado atual para outro estado

Use quando o elemento ja esta na tela e voce quer levar ele para novas propriedades.

gsap.to(".box", { x: 180, rotation: 18, duration: 1 })
to

gsap.from()

Entrada partindo de um estado inicial

Ideal para revelar conteudo: o GSAP cria o caminho do valor informado ate o CSS atual.

gsap.from(".box", { y: 80, opacity: 0, scale: .6 })
A
B
C

gsap.fromTo()

Controlar inicio e fim da animacao

Use quando voce precisa garantir o estado inicial e final, sem depender do CSS atual.

gsap.fromTo(".box", { x: -80 }, { x: 150, opacity: 1 })
fromTo

stagger

Animar varios elementos em sequencia

Perfeito para listas, galerias, cards, letras e qualquer grupo repetido.

gsap.from(".item", { y: 30, opacity: 0, stagger: .08 })

eases

Mudar a sensacao do movimento

`power`, `back`, `elastic`, `bounce`, `circ` e `expo` deixam a animacao natural ou expressiva.

ease: "elastic.out(1, .45)"

02 / Composicao

Timelines, controle e estados

gsap.timeline()

Encadear animacoes com precisao

Timeline organiza uma cena inteira e permite sobrepor animacoes com offsets como `"-=0.2"`.

tl.from(...).to(..., "-=.2").from(...)

02
03

keyframes

Varios passos em uma unica chamada

Bom para movimentos curtos com mudancas de escala, rotacao, posicao e cor.

keyframes: [{ x: 80 }, { y: -40 }, { rotate: 45 }]
KF

repeat / yoyo

Loops, pulso e animacoes continuas

`repeat`, `yoyo` e `repeatDelay` criam movimento continuo sem escrever varias chamadas.

repeat: 3, yoyo: true, repeatDelay: .1

play / pause / reverse

Controlar uma timeline

Guarde a timeline em uma variavel para tocar, pausar, reverter, reiniciar e controlar progresso.

timeline.play(); timeline.reverse(); timeline.progress(.5)

03 / Pagina real

ScrollTrigger

trigger

Revelar elementos conforme aparecem

Use `ScrollTrigger` para iniciar animacoes quando uma secao entra no viewport.

scrollTrigger: { trigger: ".secao", start: "top 75%" }

pin + scrub

Fixar uma area e sincronizar com o scroll

Com `pin` a secao fica presa enquanto a timeline avanca. Com `scrub`, o progresso acompanha o scroll.

scrub

04 / Plugins

Interacoes prontas para projetos

MotionPathPlugin

Animar seguindo um caminho SVG

Bom para mapas, linhas de produto, avatares, loading e motion graphics.

motionPath: { path: "#path", align: "#path" }

Flip

Animar entre layouts diferentes

Capture o estado, mude classes ou DOM, e deixe o Flip calcular a transicao.

const state = Flip.getState(".item"); Flip.from(state)

Util para chamadas, onboarding, banners e estados de interface.

Draggable

Arrastar elementos com fisica simples

Arraste o bloco dentro da area. Em projetos reais, combine com `bounds`, `snap` e timelines.
